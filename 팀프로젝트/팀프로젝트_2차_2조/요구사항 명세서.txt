REQ-001. 메인페이지
 - 우측 상단에 로그인 UI
 - 상단 배너(로고 및 메뉴바)
 - 각 메뉴 부분을 미리보기 제공
 - 외부링크(유튜브 혹은 스폰서 등등)

REQ-002. 로그인페이지
 - 아이디/비밀번호 입력
 - 회원가입, 로그인 버튼

REQ-003. 회원가입페이지
 - 아이디/비밀번호(중복확인), 별명, 휴대폰번호, 이메일, 지역선택
 - 회원가입 버튼
 - 관리자는 1000000번대, 이외 회원은 2000000번대 이후 UID 사용

REQ-004. 공지사항
 - 공지글 작성 기능(관리자만)
 - 로그인 필수

REQ-005. 커뮤니티
 - 로그인 안되있으면 로그인페이지 호출
(1)게시글 작성/수정/삭제
 - 본인 게시글만 수정,삭제 가능
 - 작성일자+시간/분
 - (우선순위 보통)이미지,동영상 첨부 기능
 - (우선순위 낮음)추천,조회수 기능 : 향후 재협의
 - (우선순위 낮음)하이퍼링크 걸기 기능 : 향후 재협의

(2)덧글 작성/수정/삭제
 - 본인 덧글만 수정,삭제 가능
 - 작성일자+시간/분
 - 최대 500자
 - (우선순위 낮음)하이퍼링크 걸기 기능 : 향후 재협의


REQ-006. 기록페이지
(1)부위별 운동법 소개
 - 등,가슴,어깨,팔,하체,유산소 부위별 운동법 소개
 - 각 부위별 추천 운동법 관련 유튜브 링크
(2)식단 기록
 - 대표 메뉴 및 칼로리(목록 선택 형태) 정리 후 합계 계산
 - BMI 및 권장 칼로리 대조
